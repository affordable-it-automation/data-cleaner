# Data Cleaner (Python)

## What it does
Cleans and formats messy CSV or Excel data into a consistent, usable format.
Removes duplicates, fixes formatting, and standardizes data.

## Example
Input: messy_data.csv  
Output: clean_data.csv

## Why it's useful
Messy data slows everything down and leads to errors.  
This script quickly removes duplicates, fixes formatting, and organizes data into a clean structure.

## How to run
python data_cleaner.py
